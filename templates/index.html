Comment utiliser les entités ?

Crée ici une SCI ou indivision avec sa valeur globale et sa dette.
Ensuite dans Positions, sélectionne l'entité et renseigne le % de propriété de chaque personne.
Le net attribué = (Actif brut × % propriété) – (Dette × % dette).

Référentiel

Modifiez les listes et valeurs utilisées dans toute l'application. Les changements prennent effet après enregistrement.

Import / Export

Importer depuis Excel

Importe les onglets Positions et Flux depuis ton fichier .xlsx.

Importer depuis JSON

Restaure un export JSON précédent. Les positions déjà présentes ne sont pas dupliquées.
Les allocations cibles sont restaurées si incluses.

Exporter en JSON

Télécharge toutes tes données (positions, flux, entités, cibles) en JSON.

Mode démo
